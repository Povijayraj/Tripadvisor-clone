# TripAdvisor Clone Website

A responsive TripAdvisor clone website built using HTML and CSS. This project recreates the basic design and layout of the popular travel platform with a clean and modern user interface. It includes sections like navigation bar, search section, travel destinations, hotel recommendations, and footer.

## Features
- Responsive web design
- Clean and modern UI
- Travel destination cards
- Navigation bar and footer
- Built using pure HTML and CSS

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to practice front-end web development skills and improve responsive website design techniques.

## How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
